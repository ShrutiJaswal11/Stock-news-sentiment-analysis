# Stock News Sentiment Analysis

This project performs sentiment analysis on financial news headlines related to companies such as Amazon, Google, and Meta.

The system collects news headlines using web scraping and classifies them as positive, neutral, or negative using machine learning models.

## Technologies Used

- Python
- Pandas
- NumPy
- Scikit-learn
- NLTK
- BeautifulSoup
- Matplotlib

## Machine Learning Models

- Naive Bayes
- Logistic Regression
- Support Vector Machine (SVM)

## Files in the Project

realtime.ipynb – performs real-time sentiment prediction using trained models  
train_model.ipynb – trains machine learning models  
financial-news-dataset.csv – dataset used for training  
*.pkl – trained machine learning models
